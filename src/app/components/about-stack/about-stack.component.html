The stack we used

Developing the app

Here is the tech stack that has been used to build and run this application.

Client side front-end

  1. Angular Framework using HTML, CSS and TypeScript (a superset of JavaScript).
  2. Bootstrap Framework.

Server side back-end

  1. The web application server:
    • Is written in Java using Spring Framework.
    • Is using RESTful APIs.
    • Was tested using Swagger UI.
  2. Application Dependencies: Spring Boot, Spring Web, Spring Data JPA, MySQL Driver, PostgreSQL Driver, Lombok, Spring-boot-devtools, springfox-swagger-ui, Awaitility.
  3. Database: MySQL and PostgreSQL.

The Application is:

  1. Handling HTTP requests and responses between the front-end and the back-end.
  2. Implementing CRUD - Create, Retrieve, Update and Delete Companies, Customers, Coupons & Images in a persistent storage.
  3. Sending information between the front-end and the back-end in JSON format.
